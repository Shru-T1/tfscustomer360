# Toyota Customer 360

## ML Recommendation Dashboard
A React-based customer analytics dashboard built for Toyota Financial Services' Customer 360 initiative, developed as a senior undergraduate capstone project in a team of five.
Overview
This project provides a comprehensive customer insights and analytics interface powered by machine learning-driven recommendations. It was built on a cloud-based NoSQL backend at a time when cloud NoSQL adoption in enterprise environments was still early-stage.
My Role
Co-built the frontend dashboard and contributed to the NoSQL data layer. Responsible for data visualization components and integration with the ML recommendation backend.
Tech Stack

## Frontend: 
### React 17, Reactstrap, Bootstrap 4
 - Data Visualization: D3.js, Chart.js
 - Styling: SCSS
 - Routing: React Router DOM
 - Build Tools: Webpack via Create React App

## Features

 - Interactive customer analytics dashboard with charts and data tables
 - ML-driven customer recommendation display
 - Responsive, mobile-first design
 - Multi-page layout with authentication and admin views
